Monthly Dairy Nov.2020

很久以前就有一种强烈的冲动找个地方写一个类似于日记一样的东西，但总觉得任何SNS都会像是枷锁一样限制住我的表达，而任何的本地记录又觉得 这玩意不能让别看到 略感惋惜，一时兴起就自己找了个 blog template 在 Github 上写写吧。

这玩意的好处还有 我可以像CSDN、简书一样，自己在上面写写技术性的东西，挺好。

我想让这个空白的网站如同海德格尔的林中空地一样，我在这里展开，我在这里存在，我在这里写我想写的东西，有没有人看，别人怎么看我与我无关。我要从社交媒体里逃离，我要从他人即地狱中逃离。

回过头来看了一下感觉上面三段文字好像挺错乱的，别人会不会觉得我精神分裂啊（笑）。

不过这无关紧要，反正现代人对于SNS上其他人的生活的兴趣仅限于5段话。
